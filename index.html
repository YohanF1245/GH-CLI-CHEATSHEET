GH CLI CHEAT SHEET

GitHub CLI is an open-source command line tool that brings GitHub to your terminal. You can check the status, pull requests, issues, files, gists, and other GitHub concepts.

INSTALLATION

macOS Linux Windows
$ brew install gh $ sudo apt install gh $ winget install --id GitHub.cli

LOGIN

Launch a browser to authenticate. Alternatively, use GitHub token.
$ gh auth login $ gh auth login --with-token < mytoken.txt

BROWSE

View Current GitHub repository in the web browser. Open issues or pull request 15 Open a commit page Open main.py file at line 340. Open repository settings
$ gh browse $ gh browse 15 $ gh browse 8d7656a58112aa1487d5f9c12a1c2479ab16483a $ gh browse main.py:340 $ gh browse --settings

CODESPACE

Create a code space instance for the current repository. Make sure you have proper authentication setup. Open and initialize the VSCode on the web browser
$ gh codespace create $ gh codespace code --web

GIST

List of your Gists. Publish a 'main.py' file as a gist. View a gist in terminal using id. Delete a gist using id.
$ gh gist list $ gh gist create --public main.py $ gh gist view 8cc08f7cb7d498f426a8f7d2c1dedd56 $ gh gist view 8cc08f7cb7d498f426a8f7d2c1dedd56

ISSUES

List issues in a GitHub repository. Create an issue interactively and follow the steps. Show the status of issues that is relevant to you. View issues 1 in the terminal.
$ gh issue list $ gh issue create $ gh issue status $ gh issue view 1

PR

Create a pull request interactively. View the list of open pull requests View a pull request 2 in terminal. Review pull request 24 interactively. Merge pull request 24 on GitHub.
$ gh pr create $ gh pr list $ gh pr view 2 $ gh pr review 24 $ gh pr merge 24

REPO

Create a repository interactively Clone a repository using OWNER/REPO syntax. List repositories owned by kingabzpro Create a fork for docs repository. Viewing a repository description and readme in terminal
$ gh repo create $ gh repo clone kingabzpro/blog $ gh repo list kingabzpro $ gh repo fork github/docs $ gh repo view kingabzpro/blog

WORKFLOW

List workflow files in GitHub Actions Run the workflow file 'build.yml' at the remote default branch Select a workflow to view interactively Enable/Disable a workflow
$ gh workflow list $ gh workflow run build.yml $ gh workflow view $ gh workflow enable/disable build.yml

RUN

List recent workflow runs Select a run to view interactively Watch a run until it completes Cancel a workflow run Rerun a failed run
$ gh run list $ gh run view $ gh run watch 3456 $ gh run cancel 3456 $ gh run rerun 3456

STATUS

View summary of your work on GitHub across all the repositories
$ gh status
